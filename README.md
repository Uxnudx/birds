# Telegram бот для проверки куки Roblox

Бот проверяет валидность куки Roblox и отправляет файл с валидными куки.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Получите токен бота от [@BotFather](https://t.me/BotFather) в Telegram

3. Откройте файл `bot.py` и замените `YOUR_BOT_TOKEN_HERE` на ваш токен:
```python
BOT_TOKEN = "ваш_токен_здесь"
```

## Запуск

```bash
python bot.py
```

## Использование

1. Отправьте боту команду `/start`
2. Отправьте txt файл с куки (каждая куки на новой строке)
3. Бот проверит все куки и отправит обратно файл с валидными куки

### Формат файла с куки:
```
cookie1_здесь
cookie2_здесь
cookie3_здесь
```

Также можно отправить одну куки в текстовом сообщении для быстрой проверки.

## Команды

- `/start` - Начать работу с ботом
- `/help` - Показать справку

## Примечания

- Проверка куки может занять некоторое время
- Бот проверяет валидность через официальный API Roblox
- Временные файлы автоматически удаляются после обработки
